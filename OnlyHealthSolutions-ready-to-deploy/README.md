# Only Health Solutions — Ready-to-deploy (Next.js + Tailwind)

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
- Import this repo in Vercel
- Framework preset: Next.js (auto)
- Build command: `npm run build` (auto)
- Output: Next.js default (auto)

## Assets
Logo + favicon are included in `/public`:
- `logo.png`
- `logo@2x.png`
- `favicon.ico`

## Customize
Edit: `src/app/page.jsx`
- Update `brand.addressLine`
- Update "Licensed/Insured? (Update)" badge text
- Add your Google Reviews link to `brand.googleReviewsUrl`
